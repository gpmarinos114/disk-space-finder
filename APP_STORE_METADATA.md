## Description

A powerful and intuitive disk space analyzer for macOS. Visualize your storage usage with beautiful treemap, sunburst, and tree views. Find and remove duplicate files, old unused files, and reclaim your disk space.

**Visualize Your Storage**

• Treemap view shows files as proportional rectangles — instantly see what's taking up space
• Sunburst chart displays directory hierarchy as concentric rings
• Tree view provides a familiar file browser with powerful sorting and filtering
• Charts show top files, file type distribution, and size breakdowns

**Find and Remove Duplicates**

• SHA256 hash-based duplicate detection finds exact matches
• Quick scan mode matches by size and name for fast results
• See exactly how much space each duplicate group wastes

**Reclaim Old Files**

• Find files you haven't touched in months or years
• Filter by custom time ranges (days, months, years)
• Sort by size to find the biggest space hogs

**Track Your Usage**

• Scan history saves snapshots of your disk usage over time
• Trend charts show how your storage grows
• Compare any two scans to see what changed

**Powerful Actions**

• Open files directly from the app
• Reveal files in Finder
• Compress folders with one click
• Move files to Trash with confirmation
• Export file lists to CSV

**Beautiful and Native**

• Built with SwiftUI for a native macOS experience
• Dark and light mode support
• Drag and drop folders to scan
• Keyboard shortcuts throughout
• System notifications when scans complete

**Privacy First**

• All analysis happens locally on your device
• No data is collected, transmitted, or stored externally
• No internet connection required
• No analytics or tracking
