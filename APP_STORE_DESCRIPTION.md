# Disk Space Finder

A powerful and intuitive disk space analyzer for macOS. Visualize your storage usage with beautiful treemap, sunburst, and tree views. Find and remove duplicate files, old unused files, and reclaim your disk space.

## Features

- **Treemap Visualization** — See your files as proportional rectangles, making it easy to spot what's taking up space
- **Sunburst Chart** — View directory hierarchy as concentric rings
- **Tree View** — Traditional file browser with sorting and filtering
- **Charts** — Top files, file type distribution, and size histograms
- **Duplicate Finder** — Find and remove duplicate files to reclaim space
- **Old Files Finder** — Find files you haven't touched in months or years
- **Scan History** — Track your disk usage over time with trend charts
- **Quick Actions** — Open, reveal in Finder, compress, or trash files directly
- **Export** — Export file lists to CSV for analysis

## Requirements

- macOS 14.0 Sonoma or later
- Apple Silicon or Intel Mac

## Privacy

Disk Space Finder does not collect, transmit, or store any personal data. All scanning and analysis happens locally on your device. No data is sent to any server.
